# Reservations Landingpage (sandbox)

Minimal Next.js project scaffolded to match versions from the `reservations-frontend-rm` project in this workspace.

How to run:

```zsh
# from the project root
pnpm install
pnpm dev
```

Notes:
- Uses Next.js 16.2.0 and React 19.2.4 (same as the main frontend repo).
- Tailwind is configured (v4) — after installing dependencies, run the dev server to verify CSS.
- If you want exact parity, we can copy `src`, `i18n`, and ESLint configs from the real project.
